## Задание 1 - 5. Текстовые файлы
### Текст задачи
1. Найти разность максимального и минимального элементов.
2. Вычислить минимальный элемент.
3. Переписать в другой файл строки, начинающиеся с заданного символа.
4. Получить в новом файле все компоненты исходного файла, которые делятся на m и не делятся
на n.
5. Информация о багаже пассажира описывается массивом, где каждый элемент содержит название
единицы багажа (чемодан, сумка, коробка и т.д.) и ее массу. Дан файл, содержащий сведения о
багаже нескольких пассажиров. Выяснить, имеется ли пассажир, багаж которого состоит из
одной единицы багажа массой менее m кг.
### Алгорит решения
1. Реализован статический метод заполения файла, который создает объект SteamWriter для записи в файл, с помощью цикла генерирует случайное число в диапазоне от 0 до 100 на каждой итерации и записывает его в новую строку. После завершения цила файл закрывается.
Метод DifferentMaxMin, который находит разность между максимальным и минимальным значением файла. Открываем файла для чтения с помощью StreamReader, до тех пор пока файл не закончился считываем строку, преобразовываем ее в число и последовательно обновляем значения минимума и максимума. Возвращаем разность.
Метод вывода содержимого файла PrintFile, построчно читает файл и выводит его содержимое.
2. Реализован метод заполения файла с несколькими числами в строку через пробел. Метод поиска минимума: читаем файл построчно, каждую строку разбиваем на числа (с помощью Split), преобразуем строки в число, сравниваем и обновляем минимум. Возвращаем найденный минимум.
